Install & Run Laravel


> composer global require "laravel/installer"
> # go to your projects directory or in our case xampp/htdocs
> laravel new blog

or...


> composer create-project --prefer-dist laravel/laravel Laravel54Blog "5.4.*"
> cd Laravel54Blog
> code . # launch VS Code on the current directory
> # Open Integrated Terminal on VS Code
> php artisan serve
We will be using the second option to install a specific laravel version.
It's convenient that the command prompt is inside the editor, launch via shortcut ctrl - ~
