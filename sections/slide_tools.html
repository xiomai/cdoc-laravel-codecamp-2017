Workspace Environment

Windows, Mac or UNIX based

  • WAMP for Windows (wampserver.com/en/)
  • XAMPP (apachefriends.org/index.html
  • Apache, PHP5.6 and MySQL
  • Composer (getcomposer.org)
  • Nodejs (https://nodejs.org/dist/v8.8.1/node-v8.8.1-x64.msi)
You must have some kind of PHP development environement.
Composer for installing Laravel itself.
Nodejs for compiling assets, from SASS to CSS, minifications.

Text Editor or IDE

  • Notepad ++ (notepad-plus-plus.org/)
  • Sublime (sublimetext.com/)
  • Atom (atom.io/)
  • Visual Studio Code (code.visualstudio.com/)
Editor and IDE difference? Text editor are light while IDEs are heavy on memory so editors are suitable for a simple laptop. Intellij IDEA or phpstorm.
Using IDE would be overkill (IMO) for learning since a simple project would suffice a text editor.
